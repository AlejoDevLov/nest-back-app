
# Backend en Nest


1. Abrir docker y compose mongo db.

2. Copiar el ```.env.template``` y renombrarlo a ```.env```

3. Abrir la carpeta en la ventana de comandos y ejecutar el siguiente comando.
```
docker compose up -d
```

4. Ejecutar el siguiente comando desde la terminal donde tengamos abierta la carpeta del proyecto. Esto para poner a correr el servidor de manera local el cual se ejecutar√° regularmente en el puerto 3000
```
npm run start:dev
```
